AI-Powered Travel Itinerary Planner

Your personal travel assistant, powered by AI.

About the App

This application helps users create custom travel plans using advanced AI algorithms to analyze preferences and provide tailored suggestions.

Features

  • Generative AI-driven itinerary customization
  • Integrated with social platforms for trip sharing
  • Hosted on AWS for reliability and scalability

Technology Stack

  • Front-End: React.js with Vite
  • Server Framework: NestJS
  • API Layer: GraphQL
  • ORM: Prisma
  • Database: PostgreSQL
